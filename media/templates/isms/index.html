About

ISA is a web system based on Django developed to evaluate the information security in an organization using the controls from the ISO 27002 standard. You can use ISA as an indicator during the implementation process of an ISMS. It helps to identify the controls that need to be handled according with the project scope and/or with the organization's goals.
